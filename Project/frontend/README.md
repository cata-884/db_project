# MovieApp Frontend (React + Vite)

Minimal frontend pentru proiectul de BD (filme). Ruleaza pe `http://localhost:5173` si foloseste proxy pentru API.

## Cerinte
- Node.js 18+
- Backend Spring Boot pe `http://localhost:8080`

## Rulare rapida
```bash
cd Project/frontend
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Flow de test
1. Login cu `client1 / parola1`
2. Vezi filme si detalii
3. Vizioneaza film (inregistrare)
4. Adauga recenzie
5. Profil: recomandari PL/SQL
